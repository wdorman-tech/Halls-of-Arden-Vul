## OSRIC Base Encumbrance

Encumbrance is the total weight you're carrying. It determines how fast you can move.

You have an **encumbrance allowance**, which depends on your strength score (see Table 1.1.2a). This is the amount you can carry without penalty. If you're carrying weight above your encumbrance allowance, then your maximum [[Movement Rate\|Movement Rate]] is reduced, and it's harder for you to surprise creatures you meet.

### Weight Notation

Most items have a weight in pounds. A few notations:
- **"--"**: weight doesn't add to encumbrance (very light items, worn clothing)
- **"N/A"**: has weight but isn't normally carried (e.g., horses)
- **"0 (5)"**: when worn, doesn't add to weight; if carried as extras, adds the parenthetical weight

### Coins

Ten coins weigh one pound. All coins weigh the same regardless of denomination — a "coin" is a unit of weight equal to 0.1 lbs. See [[Money\|Money]].

## Slot System (Player Doc / Arden Vul)

In addition to the OSRIC weight system, the Halls of Arden Vul Player Doc uses a slot-based encumbrance system for quick adjudication:

- **Standard items:** 1 slot each
- **Large items** (claymore, medium/large shield): 2 slots
- **Very large items** (tent, cooking pot): 3 slots
- **Tiny / negligible items**: 0 slots (e.g., chalk, pin)
- **Worn clothing / unworn-no-AC items:** 0 slots

## Effect on Movement & Surprise

The effect of encumbrance on movement and surprise is described in Table 1.5.3.3A under [[Movement Rate\|Movement Rate]]. Encumbered movement reduces speed (Full -> Three-Quarter -> Half -> One-Quarter).

In addition to encumbrance, certain armour types have an enforced [[Movement Rate\|Movement Rate]] cap regardless of carried weight (see [[Armour Class\|Armour Class]] table).
