## Spell Stats

- **Class:** [[Cleric\|Cleric]] / [[Magic-User\|Magic-User]]
- **Level:** 3rd (Cleric) / 2nd (Magic-User)
- **School:** [[Divine Divination\|Divine Divination]] / [[Arcane Divination\|Arcane Divination]]
- **Range:** Caster (Cleric) / 20ft per MU level (Magic-User)
- **Duration:** 1 round per cleric level / 1 round per MU level
- **Area of Effect:** Path 10ft wide originating from caster, length of 60ft plus 10ft per cleric level (Cleric) / See below (Magic-User)
- **Components:** V, S, M
- **Casting Time:** 10 rounds (Cleric) / 2 segments (Magic-User)
- **Saving Throw:** None

## Notes

When this spell is cast, the caster visualises an object. This can be general ("a staircase") or specific ("my shield"). If an object fitting the description is in the pathway-shaped area of effect (usually the caster turns around in a circle once the spell is cast), they will learn the exact distance of the nearest object fitting the description. Once an object is located, the spell ends.

## Details

Opaque materials don't interfere with the spell, so the caster can perceive a target object even through solid rock.

The target must be an object, not a creature or a place.

If there are several target objects within the area of effect, the spell only locates the nearest one. The spell doesn't locate objects unless they're on the same plane as the caster.

The spell tells the caster the object's location, but doesn't tell them anything about the terrain (or creatures) between them and the object. A high-level caster might detect an object that is several rooms and hallways distant, and the exact path required to reach the object's room will not be obvious.

The caster can't cast this spell unless they can visualise the target object. For the purposes of this spell, the caster can visualise the target object if they're familiar with all three of its form, function, and material.

The caster must concentrate on maintaining the spell. If the caster's [[Concentration\|Concentration]] is broken, the spell ends.
