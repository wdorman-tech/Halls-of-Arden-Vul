## Notes

When a spell or effect calls for a **System Shock check** — usually when a character is polymorphed into another bodily form, petrified and restored, or otherwise dramatically transformed — the character must roll **lower than or equal to** the number shown on a d100 (from the [[Constitution\|Constitution]] table). If the roll fails, the character does not survive the shock.

## Key Facts

- **Roll:** d100, must be ≤ System Shock number from the [[Constitution\|Constitution]] table
- **Triggers:** polymorph, stoning/restoration, certain magical transformations
- **Modifier source:** Constitution score
- **Distinct from:** [[Resurrection Success\|Resurrection Success]] (for raising the dead)
