## Notes

When travelling overland, characters move based on terrain and their [[Movement Rate\|Movement Rate]].

## Key Facts

| Description | Speed | Notes |
|---|---|---|
| Hiking, level terrain | MR x 0.2 miles/day | |
| Hiking, rugged terrain | MR x 0.15 miles/day | |
| Hiking, very rugged terrain | MR x 0.10 miles/day | |
| Forced March | up to 2x hiking speed | See [[Forced March\|Forced March]] |
| Outdoor Combat | Same as underground combat | GM may multiply by 3 for realism |

- **Mounted travel:** use the mount's MR
- **Example:** a character with MR 120 can hike 120 x 0.2 = 24 miles/day in level terrain
- **Escape pursuit:** outdoor escape uses a separate %-based system; see Section 1.5.3 in OSRIC
