## Concept

A two-handed weapon requires both hands to wield, which means **the wielder cannot also carry a shield**. Many large or heavy weapons are exclusively two-handed, while others may be wielded one-handed by sufficiently strong characters.

## Weapons That Are Two-Handed Only

- [[Halberd\|Halberd]]
- [[Two-Handed Sword\|Two-Handed Sword]]
- [[Pole Arm\|Pole Arm]] (and variants)
- [[Quarterstaff\|Quarterstaff]]
- [[Long Bow\|Long Bow]]
- [[Short Bow\|Short Bow]]
- [[Composite Long Bow\|Composite Long Bow]]
- [[Composite Short Bow\|Composite Short Bow]]
- [[Crossbow, Light\|Crossbow, Light]]
- [[Crossbow, Heavy\|Crossbow, Heavy]]

## Weapons Optionally Two-Handed (One-Handed with Min STR)

| Weapon | One-Handed STR Required |
|---|---|
| [[Battle Axe\|Battle Axe]] | 15+ |
| [[Bastard Sword\|Bastard Sword]] | 15+ |
| [[Broad Sword\|Broad Sword]] | 12+ |
| [[Flail, Heavy\|Flail, Heavy]] | 14+ |
| [[Mace, Heavy\|Mace, Heavy]] | 13+ |
| [[Morning Star\|Morning Star]] | 16+ |
| [[Pick, Heavy\|Pick, Heavy]] | 14+ |
| [[Trident\|Trident]] | 14+ |
| [[Warhammer, Heavy\|Warhammer, Heavy]] | 15+ |
| [[Spear\|Spear]] | (1 or 2)|

If the strength requirement is not met, the weapon may still be used one-handed but suffers the [[Weapon Proficiency\|non-proficiency penalty]] for the character's class.
