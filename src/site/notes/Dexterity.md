## Notes

Dexterity covers more than fine motor skills — it includes speed, agility, hand-eye coordination, and quick wits that improve reaction time. A high-Dexterity character could be a hooded [[Thief\|Thief]], a swashbuckler with a rapier, or a stealthy halfling slinger. Dexterity affects accuracy with missile weapons, your armour class (chance to dodge), missile-combat initiative, and agility-based saving throws.

It is the prime requisite for [[Thief\|Thief]] and [[Assassin\|Assassin]] and modifies thief skills.

## Key Facts

- **Range:** 3-18 (up to 19 by magical means)
- **Generation:** 3d6 in order, with one swap — see [[Modified Character Creation\|Modified Character Creation]]
- **Prime requisite for:** [[Thief\|Thief]], [[Assassin\|Assassin]]
- **Modifiers (condensed):**
  - 3: -3 surprise, -3 missile to hit, +3 init segment, AC +4 [-4], -4 agility save
  - 4: -2 / -2 / +2 init / +3 [-3] / -3
  - 5: -1 / -1 / +1 init / +2 [-2] / -2
  - 6: 0 / 0 / 0 / +1 [-1] / -1
  - 7-14: no modifiers
  - 15: AC -1 [+1], +1 agility save
  - 16: +1 surprise, +1 missile to hit, AC -2 [+2], +2 agility save
  - 17: +2 surprise, +2 missile to hit, -1 init segment, AC -3 [+3], +3 agility save
  - 18: +3 surprise, +3 missile to hit, -2 init segment, AC -4 [+4], +4 agility save
  - 19: +3 surprise, +3 missile, -3 init segment, AC -4 [+4], +4 agility save
- **AC adjustment** uses descending [or ascending] AC depending on the table.
- **Initiative effect on missile combat** lets high-DEX missile users act earlier.
- **Saving throw modifier (agility):** applies vs aimed magic items, area-of-effect spells (e.g. fireball), and any save where physical agility is the key factor.
- Modifies [[Thief\|Thief]] skills.
